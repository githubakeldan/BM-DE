# BM-DE v1

## Proje

BM-DE (BookMind Development Environment), BookMind platformunun temel altyapısını sağlayan Docker tabanlı geliştirme ortamıdır.

## Amaç

BM-DE'nin amacı BookMind'ın tüm servislerini tek bir platform altında çalıştırmaktır.

## Temel Servisler

- Ollama
- Open WebUI
- PostgreSQL
- Redis
- Qdrant
- n8n

## BookMind Servisleri

- BM-Core
- BM-Learn
- BM-Agent
- BM-RAG
- BM-Voice
- BM-Console

## Yönetim

Tüm servisler Docker Compose ile yönetilir.
